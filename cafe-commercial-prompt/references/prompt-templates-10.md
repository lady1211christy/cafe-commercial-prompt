# 10 種版型 Prompt 模板規則

下列模板是 prompt 結構規則，不是要逐字照抄，而是要依照使用者提供的飲料名稱、材料、參考圖，自動替換內容。

## 01｜垂直爆炸材料分解海報
Prompt 必須包含：
- 成品飲料在底部中央
- 材料沿垂直軸層層向上漂浮
- 至少點出 4-8 個材料
- 有標示線與食譜海報感
- 背景為米色、暖灰、咖啡色系的高級編輯風

## 02｜材料環繞型產品定裝圖
Prompt 必須包含：
- 成品飲料居中
- 材料均衡分布於四周
- 靜物商攝感
- 桌面乾淨
- 適度出現杯器、碗皿、冰塊、果實、糖漿器皿

## 03｜雜誌封面人物飲品圖
Prompt 必須包含：
- 人物與飲料互動
- 雜誌封面文字排版
- 封面大標題
- premium lifestyle editorial feeling
- 光線柔和，畫面高級

## 04｜自然戶外飲品商品照
Prompt 必須包含：
- 單杯飲料主體
- 木桌或自然材質平台
- 綠葉虛化背景
- 自然陽光
- 咖啡館戶外氛圍

## 05｜動態飛濺飲品廣告圖
Prompt 必須包含：
- 單杯飲料主體
- 咖啡液或糖漿飛濺
- 高速瞬間感
- 廣告質感強
- 光影戲劇化但仍乾淨高級

## 06｜簡約材料盤點食譜海報
Prompt 必須包含：
- 成品飲料與材料分散擺盤
- 大標題與簡約內文區塊
- 做法 / 材料區塊
- 清楚留白
- 編輯感食譜海報風格

## 07｜手繪插畫風食譜教學頁
Prompt 必須包含：
- 插畫風成品飲料
- 插畫版材料圖示
- 分步驟教學格
- 手寫感文字與裝飾
- 溫暖可愛日系手帳氛圍

## 08｜攝影步驟型教學版面
Prompt 必須包含：
- 成品主圖
- 6 至 9 張步驟照片格
- 材料區塊
- 教學頁版面
- 簡潔清楚的編排

## 09｜品牌海報 / 廣告主視覺
Prompt 必須包含：
- 單杯飲料 hero shot
- 大標題、品牌語、簡短品項資訊
- 海報感 typography
- 高級時尚品牌感
- 光影好看、產品精緻

## 10｜咖啡品牌官網首頁首屏
Prompt 必須包含：
- 首屏 hero 圖
- 網站導覽列
- 主標題與副標題
- CTA 按鈕
- 產品介紹模組或下方內容區塊
- 咖啡品牌網站 mockup 感
