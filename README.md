# 飲品商攝 10 版型 Prompt Skill

把一張飲料圖片，快速轉成指定風格的生圖 Prompt。

這個 Skill 會先用中文引導使用者：
1. 上傳 1 張飲料圖片  
2. 輸入飲料名稱  
3. 輸入調飲素材 / 材料  
4. 從 10 種圖像風格中選 1 種  

接著直接輸出完整的 Prompt 套組，方便拿去生成圖片。

---

## 這個 Skill 可以做什麼？

你可以用它把飲料圖片轉成以下用途的 Prompt：

- 飲品商攝圖
- 材料拆解圖
- 食譜教學頁
- 雜誌封面風
- 品牌海報
- 官網首頁視覺

它不是直接幫你生圖，  
而是幫你把最麻煩的 Prompt 整理好，讓你可以快速接到 ChatGPT、Midjourney 或其他生圖工具使用。

---

## 使用方式

請先提供以下 4 項資訊：

1. 上傳 1 張飲料圖片  
2. 飲料名稱  
3. 調飲素材 / 材料  
4. 從 10 種圖像風格中選 1 種  

你可以直接用這個格式輸入：

```text
飲料名稱：椰香牛奶冰咖啡
材料：咖啡、鮮奶、椰子水、椰肉、糖漿、冰塊
想做的風格編號：2
